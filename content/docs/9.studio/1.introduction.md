## 新的 Nuxt Studio 模块

当 NuxtLabs 加入 Vercel 时，团队承诺将 [nuxt.studio](https://nuxt.studio) 发布为免费的、开源的、可自托管的 Nuxt 项目模块。

在 GitHub 上发现 Nuxt Studio 模块。

这个承诺现在已经实现。得益于 Vercel 的支持，**Nuxt Studio 模块**的第一个版本已经可用。

您现在可以**直接在生产环境中编辑内容**，具有实时预览和 GitHub 集成功能，所有这些都在您自己的 Nuxt 应用程序内完成。

## 它与独立平台有何不同？

最初作为 [nuxt.studio](https://nuxt.studio) 上的托管平台提供，Studio 已演变为一个开源的 Nuxt 模块，您可以将其与您的 Nuxt Content 网站一起部署。

这意味着内容编辑者可以直接在生产环境中管理和更新内容，无需本地开发工具或 Git 知识。

### 主要区别

- ✅ **自托管** - 完全在您自己的基础设施上运行
- ✅ **无外部依赖** - 不需要 API 或第三方服务
- ✅ **免费开源** - 在 MIT 许可下发布
- ✅ **内置集成** - 在您的 Nuxt 应用程序内工作

::warning
新的 Nuxt Studio 模块需要一个服务器端路由进行身份验证。
虽然静态生成仍然受支持，但您的网站必须**部署在支持服务器端渲染（SSR）**的平台上。
::
