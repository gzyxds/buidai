Prose 组件是 HTML 排版标签的替代品。Prose 组件提供了一种简单的方式来自定义内容 UI。

要覆盖 prose 组件，请在项目的 `components/content/` 目录中创建一个同名组件（例如：`components/content/ProseA.vue`）。

::note
Prose 组件最初是 [`@nuxtjs/mdc`](https://github.com/nuxt-modules/mdc) 的一部分。
::

## `ProsePre`

::code-group
````md [代码]
  ```js [file.js]{2} meta-info=val
  export default () => {
    console.log('代码块')
  }
  ```
````

  :::code-preview{icon="i-lucide-eye" label="预览"}
  ```js [file.js]
  export default () => {
    console.log('代码块')
  }
  ```
  :::
::

组件属性将是：

```json
{
  code: "export default () => {\n    console.log('代码块')\n}"
  language: "js"
  filename: "file.js"
  highlights: [2]
  meta: "meta-info=val"
}
```

查看[高亮选项](/docs/getting-started/configuration#highlight)以了解更多关于语法高亮的信息。

::callout{type="warning"}
如果您想在文件名中使用 `]`，您需要用 2 个反斜杠来转义它：`\\]`。这是必要的，因为 JS 会自动转义字符串中的反斜杠，所以 `\]` 将被解析为 `]`，破坏我们的正则表达式。
::
